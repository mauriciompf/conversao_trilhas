# CONVERSÃO DE TRILHAS

## Instalação

### Pré-requisitos

- [Node.js](https://nodejs.org/en)
- [NPM](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
- [PNPM](https://pnpm.io/)

### Passo a Passo (clone)

```bash
# Clone o repositório
git clone https://github.com/mauriciompf/conversao_trilhas

# Entre no diretório
cd conversao_trilhas

# Instale as dependências
pnpm install

# Verifique todos os branches (locais e remotos)
git branch -a

# Faça checkout para o branch develop
git checkout develop

# Puxe as mudanças mais recentes do branch develop
git pull origin develop

```
